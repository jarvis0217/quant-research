## 一、研究目标

对A股全市场进行量化选股回测，将选股池从银行板块扩展到全市场（5000+只），对比趋势动量、超跌反弹、低波动、多因子等策略，找出真正有alpha的方向和最优参数。

---

## 二、数据处理

### 2.1 选股池筛选

| 筛选条件 | 说明 |
|----------|------|
| 月均成交量 > 阈值 | Top300活跃股（阈值约2亿股/月） |
| 上市时间 > 24个月 | 排除新股 |
| 实际入选 | 300只 |

**注意**: 数据库 `amount` 字段99.8%为NULL，必须用 `vol`（成交量）过滤。

### 2.2 因子计算（月频截面）

| 因子 | 说明 |
|------|------|
| ret_1m/3m/6m/12m | 过去N月收益率 |
| vol_12m | 12月收益波动率（年化） |
| rs_12m | 12月相对强弱（价格位置） |

因子在月末截面计算，持仓在月间用日线数据计算净值。

## 三、策略配置

### Alpha公式

```
A.趋势动量: +0.30*z_ret_3m +0.25*z_ret_6m +0.20*z_ret_12m -0.10*z_vol_12m
B.超跌反弹: -0.30*z_ret_1m -0.20*z_rs_12m -0.15*z_vol_12m
C.低波动:   -0.35*z_vol_12m +0.25*z_rs_12m -0.10*z_ret_1m
D.多因子:   +0.15*z_ret_3m +0.15*z_ret_6m -0.15*z_vol_12m -0.10*z_ret_1m -0.10*z_rs_12m
```

## 五、关键结论

1. **趋势动量最优**: 3月+6月动量权重最高（0.30+0.25），叠加12月趋势确认
2. **适度集中更佳**: Top10风险收益比最佳（夏普2.77），Top5回撤过大（-32.9%）
3. **超跌反弹失效**: 2023-2026整体下跌趋势中，抄底策略（负动量+高RS）亏损
4. **低波动防御**: 胜率最高但收益有限，适合熊市配置
5. **选股池局限**: 实际为Top300而非全市场，含部分指数代码，结果参考性有限
